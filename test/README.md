#### Raftee Client Side ####

Technology Needed:
Node.js
Node Package Manager (NPM)
Yo - http://yeoman.io
Grunt - http://yeoman.io
Bower - http://yeoman.io

# Installing Yo, Grunt, Bower
npm install -g yo grunt-cli bower

##### To Build Application run:
grunt

##### To start web server that will serve up the client run:
grunt server
